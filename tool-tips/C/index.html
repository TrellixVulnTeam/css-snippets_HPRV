Quick tips

I am a simple CSS3-powered tooltip.

I can be on top, on the right,
left and, of course, on the bottom.

Add CSS from this pen, the markup will look like this:

<i class="qtip tip-top" data-tip="Tip content">
Any text / image / element
</i>

Available options: .tip-top, .tip-left, .tip-right or .tip-bottom
